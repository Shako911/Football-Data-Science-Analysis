# ⚽ Football Data Science Analysis (ბილეთების ფასი და მატჩის შედეგის პროგნოზირება)

ეს პროექტი წარმოადგენს ორ დამოუკიდებელ მანქანური სწავლების (ML) ამოცანას, რომელიც მიზნად ისახავს საფეხბურთო კლუბისთვის სტრატეგიული გადაწყვეტილებების მიღებას მონაცემებზე დაყრდნობით. პროექტი აგებულია Python-ისა და `scikit-learn`-ის გამოყენებით.

## 🎯 ამოცანები და მიზნები

პროექტი ორ მთავარ მიმართულებას იკვლევს: რეგრესიას (ფასწარმოქმნა) და კლასიფიკაციას (შედეგის პროგნოზი).

### 1. ამოცანა 1: ბილეთების ფასწარმოქმნა (რეგრესია)

* **მოდელი:** წრფივი რეგრესია (Linear Regression)
* **მიზანი:** იმის პროგნოზირება, თუ რა უნდა იყოს მატჩის ბილეთის ოპტიმალური ფასი სხვადასხვა ფაქტორების (მაგ., გუნდის რეიტინგი, მოთხოვნა) გათვალისწინებით.
* **შედეგები:** მოდელმა მიაღწია $R^2 = 0.7202$ სიზუსტეს, რაც ნიშნავს, რომ ის ხსნის ფასის ვარიაციის 72%-ს.
* **მთავარი ინსაითი:** ფასზე ყველაზე დიდი გავლენა აქვს `Match_Importance`, `Historical_Demand` და `Opponent_Rating` ფაქტორებს.

### 2. ამოცანა 2: მატჩის შედეგის პროგნოზირება (კლასიფიკაცია)

* **მოდელი:** ლოგისტიკური რეგრესია (Logistic Regression)
* **მიზანი:** მატჩის შედეგის (მოგება/წაგება) პროგნოზირება ისეთი ფაქტორების გამოყენებით, როგორიცაა `Team_Rating`, `Injuries`, `Rest_Days` და ა.შ.
* **შედეგები:** მოდელმა აჩვენა მაღალი სიზუსტე (Accuracy) და სანდოობა (ROC-AUC) (მაგ., Accuracy = 93.75%, ROC-AUC = 0.9969).
* **მთავარი ინსაითი:** მოგების ალბათობაზე ყველაზე დიდი გავლენა აქვს `Rating_Diff`, `Team_Rating`-სა და `Opponent_Rating`-ს.
